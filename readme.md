# AllShoes
<div align = center >
<p>El cliente ha proporcionado cuatro diapositivas en formato de imagen (mockups) que representan 
el diseño que desea para su presentación en línea. Su misión será convertir estos mockups en 
páginas web interactivas. Cada diapositiva se convertirá en una página web separada que formará 
parte de la presentación final. Además se les proporciona unos videos donde pueden ver el 
comportamiento que debe tener la página creada.</p>
  
## Descripción del Proyecto

 <p>Este proyecto consiste en el desarrollo de una página web para una tienda de zapatos online.  
La página permite a los usuarios conocer los productos disponibles, información sobre la tienda y formas de contacto.</p>

## Navegación por la Web

<p>La página incluye un menú en el header que se encuentra en todas las páginas, estas son:</p>

**Inicio** → Página principal del sitio.

**Tienda** → Muestra los modelos de zapatos disponibles.

**Nosotros** → Información sobre la empresa y sus redes.

**Contacto** → Formulario o información para contactar con la tienda y dejar mensajes.
